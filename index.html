Responsive Webdesign


Responsive web design is a web development approach that creates dynamic changes to the appearance of a website, depending on the screen size and orientation of the device being used to view it. Responsive web design uses CSS media queries to dynamically change the page layout based on the target device such as display type, width, height, etc.
